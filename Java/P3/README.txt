For this project we had to read in points and edges from a file and be able to calculate and graph the shortest path between the two points, a mapping application. The application draws in multiple past labs such as the shortest path and graph creating labs. The runtime for the shortest path is big Oh (E log(V)).

Compile: javac *.java
run: java Mapping map.txt -show -directions start end
